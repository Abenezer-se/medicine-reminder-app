## Description

The Figma design includes the full UI/UX of the application, covering all major screens and user flows.  
It is designed with a clean and user-friendly interface to help users manage their medicine schedules effectively.
 
## Screens Included

- Splash Screen
- Onboarding Screens
- Home Screen
- Add Medicine Screen
- Health Screen
- Symptoms Screen
- History Screen
- Pharmacies Screen
- Settings Screen
- Emergency Screen
 
## Design Details

- Primary Color: Green (#2E9E8E)
- Style: Minimal, modern, and user-friendly
- Platform: Mobile (Android & iOS)
