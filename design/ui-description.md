# UI Description – Medicine Reminder App (MedAlarm)

This document describes the visual design, layout style, and user experience of the Medicine Reminder mobile application.


## 1. Design Style

The application uses a clean, simple, and modern design.

- Minimal interface (no unnecessary elements)
- Easy to understand for all users
- Focus on clarity and usability


## 2. Color Scheme

- Primary Color: Green (#2E9E8E)
- Background Color: White
- Text Color: Dark Gray / Black

Reason:
Green represents health, safety, and trust, which fits a medicine-related application.


## 3. Typography

- Simple and readable font
- Large headings for important information
- Clear labels for inputs and buttons

Purpose:
To ensure users can easily read and understand the content.


## 4. Buttons and Icons

- Rounded buttons for a modern look
- Clear icons used for:
  - Medicine
  - Health
  - Notifications
  - Emergency

Button Types:
- Primary buttons: Green (main actions like Save, Add)
- Secondary buttons: Light color

## 5. Navigation

- Side navigation drawer is used
- Allows easy access to:
  - Home
  - Medicines
  - Health
  - History
  - Settings
  - Emergency

Purpose:
To make navigation simple and fast.


## 6. User Experience (UX)

The app is designed to be:

- Easy to use
- Fast to navigate
- Suitable for all age groups

Key UX features:
- Simple form to add medicine
- Clear reminder notifications
- Easy tracking of medicine history


## 7. Layout Consistency

- Same color scheme across all screens
- Same button styles
- Consistent spacing and alignment

Purpose:
To provide a smooth and predictable experience.

## 8. Responsiveness

The app is designed for mobile devices (Android and iOS).

## 9. Accessibility

- Good color contrast
- Readable text size
- Buttons are easy to tap

## Conclusion

The UI design focuses on simplicity, clarity, and usability to help users manage their medicine reminders effectively.
