
 # Wireframes – Medicine Reminder App (MedAlarm)

This document describes the layout and structure of the main screens of the Medicine Reminder mobile application.  
The wireframes are based on the UI/UX design created in Figma.

## 1. Splash Screen

Description:
- Displays app logo (MedAlarm)
- Shows slogan: "Take Your Medicine on Time"

Action:
- Automatically navigates to onboarding screens
 
## 2. Onboarding Screens

### Screen 1 – Reminder Introduction
Description:
- Icon: Medicine capsule
- Title: "Never Miss a Dose"
- Short description about reminders
- "Next" button
- "Skip" option

Action:
- Next → Go to next onboarding screen
- Skip → Go to Home Screen
 
### Screen 2 – Health Tracking
Description:
- Icon: Heart
- Title: "Track Your Health"
- Description about monitoring health

Action:
- Next → Next screen
- Back → Previous screen
 
### Screen 3 – Symptom Checker
Description:
- Icon: Stethoscope
- Title: "Symptom Checker"

Action:
- Next → Next screen
- Back → Previous screen
 
### Screen 4 – Find Pharmacies
Description:
- Icon: Location pin
- Title: "Find Pharmacies"

Action:
- Next → Enter app
- Back → Previous screen
 
## 3. Navigation Drawer (Menu)

Description:
- Side menu with options:
  - Home
  - Medicines
  - Health
  - Symptoms
  - History
  - Pharmacies
  - Settings
  - Emergency

Action:
- User selects any section to navigate
 
## 4. Home Screen

Description:
- Greeting message (e.g., Good Morning)
- Summary cards:
  - Active medicines
  - Taken today
  - Pending
  - Skipped
- Today’s schedule list
- Floating "Add" button

Actions:
- Add → Open Add Medicine screen
- Click medicine → Mark as taken/skipped
 
## 5. Add Medicine Screen

Description:
- Input fields:
  - Medicine name
  - Dosage
  - Frequency
  - Time
  - Start date
  - End date
- "Add Time" button
- "Done" button

Action:
- Save medicine and create reminder
 
## 6. Health Screen

Description:
- Displays:
  - Blood pressure
  - BMI value
- BMI calculator:
  - Height input
  - Weight input
- Result display (Normal/Underweight/Overweight)

Action:
- User inputs data → Click "Calculate BMI"
 

## 7. Symptoms Screen

Description:
- Search bar
- List of common symptoms:
  - Headache, Fever, Cough, etc.
- "Analyze Symptoms" button

Action:
- User selects symptoms → Analyze
 

## 8. History Screen

Description:
- List of medicines taken
- Status labels:
  - Pending
  - Completed
- Search and filter options

Action:
- User views past records
 
## 9. Pharmacies Screen

Description:
- Search bar
- "Find" button
- "My Location" option
- Map view showing nearby pharmacies

Action:
- User searches and views pharmacy locations
 
## 10. Settings Screen

Description:
- Profile information
- Dark mode toggle
- Notification settings
- Reminder time settings
- App information

Action:
- User updates preferences
 

## 11. Emergency Screen

Description:
- Emergency alert button
- Contact list
- Quick actions:
  - Call
  - Health hotline

Action:
- User sends emergency alert
 

## 12. Add Contact (Emergency)

Description:
- Input fields:
  - Name
  - Description
  - Phone number

Action:
- Save contact
 

## Design Notes

- Primary color: Green (#2E9E8E)
- Clean and simple UI
- Focus on usability and accessibility
- Mobile-first design
