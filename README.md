# Medicine Reminder App

A cross-platform mobile application built using Flutter that helps users manage and receive reminders for their medications.

## 📱 Project Description

The Medicine Reminder App allows users to:
- Add medicines
- Set reminder times
- View scheduled medications
- Receive notifications to take medicine on time

This project is developed as part of the Mobile Application Development course.

## 🚀 Technologies Used

- Flutter
- Dart
- Android Studio
- Git & GitHub

## 🌍 Supported Platforms

- Linux
