# Feature List

1. **User Registration and Login**
   - Users can create an account with email/phone.
   - Users can log in and log out securely.

2. **Add Medicine**
   - Users can add medicine name, dosage, and schedule.
   - Option to add medicine notes or instructions.

3. **Set Reminder Time**
   - Users can set multiple reminder times per medicine.
   - Reminders can repeat daily, weekly, or custom intervals.

4. **Notification Reminder**
   - Push notifications alert users when it’s time to take medicine.
   - Notifications work even if the app is closed.

5. **Medicine History**
   - Users can view a history of taken and missed doses.
   - Optional daily, weekly, and monthly summary reports.

6. **Edit Medicine Schedule**
   - Users can update medicine details or reminder times.

7. **Delete Medicine**
   - Users can remove medicines from their list.

8. **Search and Filter Medicines**
   - Users can search by medicine name.
   - Users can filter by active, completed, or missed medicines.

9. **Map / Pharmacy Locator**
   - Users can see nearby pharmacies and clinics on a map.
   - Clicking a pharmacy shows contact info and directions.
   - Optional: Filter by open now or distance.

10. **Dark Mode / Light Mode**
    - Users can toggle app theme between dark and light mode for better readability.

11. **User Settings**
    - Users can manage notification preferences, account info, and app settings.
