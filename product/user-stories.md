# User Stories

## 1. Add Medicine
1. As a user, I want to add a medicine with its name, dosage, and schedule so I can track it.
2. As a user, I want to add optional notes to a medicine so I can include instructions from my doctor.

## 2. Set Reminder Time
3. As a user, I want to set multiple reminder times for a medicine so I don’t miss doses.
4. As a user, I want to repeat reminders daily, weekly, or custom intervals so the schedule matches my prescription.

## 3. Notification Reminder
5. As a user, I want to receive notifications when it’s time to take medicine so I never forget.
6. As a user, I want notifications to work even if the app is closed so I can rely on them.

## 4. Medicine History
7. As a user, I want to view my history of taken and missed doses so I can track adherence.
8. As a user, I want daily, weekly, and monthly summaries so I can see patterns in my medicine usage.

## 5. Edit Medicine Schedule
9. As a user, I want to update medicine details or reminder times so I can adjust to changes in my prescription.

## 6. Delete Medicine
10. As a user, I want to delete a medicine so I can remove medicines I no longer take.

## 7. Search and Filter Medicines
11. As a user, I want to search for a medicine by name so I can find it quickly.
12. As a user, I want to filter medicines by active, completed, or missed so I can manage my list efficiently.

## 8. Map / Pharmacy Locator
13. As a user, I want to see nearby pharmacies and clinics on a map so I can easily find where to get my medicine.
14. As a user, I want to click a pharmacy to see contact info and directions so I can visit or call them directly.
15. As a user, I want to filter pharmacies by open now or distance so I can choose the most convenient one.

## 9. Dark Mode / Light Mode
16. As a user, I want to toggle between dark and light mode so I can read comfortably in different lighting conditions.

## 10. User Settings
17. As a user, I want to manage notification preferences so I can control how I receive reminders.
18. As a user, I want to update my account info so I can keep my profile current.
19. As a user, I want to reset settings to default so I can fix any mistakes or changes easily.
