# User Stories

## 1. User Registration and Login
1. As a new user, I want to create an account so I can save my medicine schedule.
2. As a user, I want to log in securely so only I can access my data.
3. As a user, I want to reset my password if I forget it.

## 2. Add Medicine
4. As a user, I want to add a medicine with its name, dosage, and schedule so I can track it.
5. As a user, I want to add optional notes to a medicine so I can include instructions from my doctor.

## 3. Set Reminder Time
6. As a user, I want to set multiple reminder times for a medicine so I don’t miss doses.
7. As a user, I want to repeat reminders daily, weekly, or custom intervals so the schedule matches my prescription.

## 4. Notification Reminder
8. As a user, I want to receive notifications when it’s time to take medicine so I never forget.
9. As a user, I want notifications to work even if the app is closed so I can rely on them.

## 5. Medicine History
10. As a user, I want to view my history of taken and missed doses so I can track adherence.
11. As a user, I want daily, weekly, and monthly summaries so I can see patterns in my medicine usage.

## 6. Edit Medicine Schedule
12. As a user, I want to update medicine details or reminder times so I can adjust to changes in my prescription.

## 7. Delete Medicine
13. As a user, I want to delete a medicine so I can remove medicines I no longer take.

## 8. Search and Filter Medicines
14. As a user, I want to search for a medicine by name so I can find it quickly.
15. As a user, I want to filter medicines by active, completed, or missed so I can manage my list efficiently.

## 9. Map / Pharmacy Locator
16. As a user, I want to see nearby pharmacies and clinics on a map so I can easily find where to get my medicine.
17. As a user, I want to click a pharmacy to see contact info and directions so I can visit or call them directly.
18. As a user, I want to filter pharmacies by open now or distance so I can choose the most convenient one.

## 10. Dark Mode / Light Mode
19. As a user, I want to toggle between dark and light mode so I can read comfortably in different lighting conditions.

## 11. User Settings
20. As a user, I want to manage notification preferences so I can control how I receive reminders.
21. As a user, I want to update my account info so I can keep my profile current.
22. As a user, I want to reset settings to default so I can fix any mistakes or changes easily.
